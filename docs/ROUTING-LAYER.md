# Routing Layer

The routing layer is inspired by `hack-skills`.

It answers:

> **Which technical skill family should this task enter before selecting a specific leaf skill?**

The routing path is:

```text
phase -> method -> router -> leaf
```

Examples:

- `web` + `JWT/session` → `auth-sec` → `jwt-oauth-token-attacks`
- `web` + `GraphQL` → `api-sec` → `graphql-and-hidden-parameters`
- `reverse` + `loader behavior` → `malware-loader-analysis`
- `evasion` + `WAF/403` → `windows-av-evasion` → `waf-bypass-techniques`

# Method Layer

The method layer is inspired by `qiushi-skill` and answers one question before technical expansion:

> **How should the agent think about the current offensive task?**

Core methods:

- `investigation-first`
- `contradiction-analysis`
- `concentrate-forces`
- `overall-planning`
- `workflows`
- `practice-cognition`
- `criticism-self-criticism`

The runtime keeps this layer lightweight: it emits only a compact method tag and does not inject full doctrine text by default.

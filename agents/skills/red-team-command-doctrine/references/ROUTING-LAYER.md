# Routing Layer

The routing layer narrows:

```text
phase -> router -> leaf
```

Examples:

- web + token/auth language -> auth-sec -> jwt-oauth-token-attacks
- web + GraphQL language -> api-sec -> graphql-and-hidden-parameters
- reverse + loader behavior -> malware-loader-analysis
- evasion + WAF/403 language -> windows-av-evasion -> waf-bypass-techniques
